? Atalhos e Recursos de Produtividade no N8N

* GERAIS
Ctrl + Alt + N     → Novo workflow
Ctrl + O           → Abrir workflow
Ctrl + S           → Salvar workflow
Ctrl + Z           → Desfazer
Ctrl + Shift + Z   → Refazer
Ctrl + Enter       → Executar workflow
Ctrl + I           → Abrir lista de execuções

* CANVAS
Espaço + Arrastar  → Mover visualização
+ ou =             → Zoom in
- ou _             → Zoom out
0                  → Resetar zoom
1                  → Ajustar workflow à tela

* NODES
Tab                → Abrir painel de nodes
Enter              → Inserir node selecionado
Esc                → Fechar painel de nodes
Setas ← ↑ ↓ →      → Navegar entre nodes
Shift + ← / →      → Seleção múltipla de nodes
Ctrl + C / Ctrl + V→ Copiar / Colar node
Ctrl + X           → Cortar node
Delete             → Excluir node
D                  → Desativar node
F2                 → Renomear node
P                  → Fixar dados no node
Shift + S          → Adicionar anotação (sticky note)



No n8n, o ícone da vassourinha é o botão Clear Canvas (ou "Limpar o Canvas").

Ele serve para:

* Remover todos os nós (nodes) e conexões que estão no workflow atual.
* Funciona como um “reset” visual do espaço de trabalho.

⚠️ Atenção: ao clicar, todo o conteúdo do workflow será apagado. Se não tiver salvo antes, você perderá as alterações.
