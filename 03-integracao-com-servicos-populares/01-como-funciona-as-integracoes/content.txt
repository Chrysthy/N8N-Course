* Conceito de integração com serviços populares 

As integrações no n8n funcionam como conexões automáticas entre serviços, APIs e aplicativos, permitindo que você crie fluxos (workflows) que executam tarefas sem precisar programar tudo na mão. A lógica é simples: cada serviço vira um “node”, e você combina esses nodes para criar automações.

* Como funcionam as integrações no n8n

✅ 1. Nodes são blocos de integração

Cada node representa uma ação ou evento de um serviço.
Exemplos:

HTTP Request → fazer requisições para qualquer API
Google Sheets → ler, criar ou atualizar planilhas
Telegram → enviar mensagens automaticamente
Webhook → receber dados externos
OpenAI/Groq/LLM → gerar textos, classificar, resumir etc.
Cada integração no n8n é basicamente um node específico daquele serviço.


✅ 2. Conexões (Credentials)

Para usar uma integração que precisa de autenticação, você cria uma credential, que é onde você coloca:

API Key
Token
Client ID + Secret
Login OAuth2
Depois, você “anexa” essa credential ao node.
Isso permite que o n8n se conecte ao serviço com segurança.


✅ 3. Ações prontas + ações personalizadas

A maioria das integrações possui ações prontas:
Exemplo no node do Google Sheets:

Create Row
Update Row
Read Rows
Delete Row

Mas se algo não existir no node, você pode usar:

→ HTTP Request

Para chamar qualquer endpoint que não tenha integração pronta.
Ou seja, o n8n consegue integrar com qualquer API do mundo.


✅ 4. Encadeamento dos nodes

Você conecta um node ao outro formando uma lógica:
Webhook → Processamento → Enviar dados para um serviço
Exemplo prático:

Receber um pedido via webhook
Consultar estoque em outro sistema via HTTP Request
Registrar pedido no Google Sheets
Enviar uma mensagem no WhatsApp/Telegram
Gerar um e-mail de confirmação com Gmail
Cada saída vira a entrada do próximo node.


✅ 5. Integrações podem consumir ou emitir dados

Alguns nodes:

- Start nodes

Webhook
Cron
Trigger do Telegram
IMAP Email Trigger
Eles iniciam o fluxo quando algo acontece.

- Action nodes

OpenAI, Google Sheets, Discord, GitHub etc.

Eles recebem dados, fazem algo, e passam adiante.


✅ 6. Dados via JSON

Todos os nodes no n8n “conversam” usando JSON.
O n8n organiza tudo em:

{
  "json": {
    "name": "Chrystine",
    "email": "teste@example.com"
  }
}


Isso facilita filtrar, transformar ou reutilizar informações.


✅ 7. Transformações entre integrações

Além das conexões diretas, o n8n tem nodes de manipulação:

Set → alterar ou criar campos
Edit Fields → mudar estrutura de dados
If / Switch → lógica condicional
Merge → juntar resultados
Code (JavaScript) → customizações avançadas

Com isso, você adapta os dados antes de enviar para outra integração.
